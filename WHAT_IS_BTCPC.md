# What Is BTCPC?

## The One-Sentence Version

BTCPC is a blockchain where mining means doing useful AI work instead of solving abstract math puzzles.

## The One-Paragraph Version

Bitcoin Proof of Compute (BTCPC) is a sovereign blockchain — its own chain, not a token on someone else's platform — where miners earn BTCPC tokens by providing real AI inference to the network. Instead of dedicating electricity to SHA-256 hash puzzles that only produce security (like Bitcoin), BTCPC miners run AI models and answer real questions from real users. The energy spent mining BTCPC produces actual, useful AI output. It's the first blockchain where the work that secures the network IS the work that serves users.

## How It's Different From Bitcoin

| | Bitcoin | BTCPC |
|---|---|---|
| Mining work | SHA-256 hash puzzles (security only) | AI inference (security + useful output) |
| What miners produce | Network security | Network security + AI answers, analysis, creative writing |
| Who uses the mining output | Nobody | Real users and applications |
| Energy efficiency | Wasteful by design | Every watt produces something valuable |
| Supply | 21 million BTC | 42 million BTCPC |
| Smallest unit | 1 satoshi | 1 dream |

## How It's Different From OpenAI / Anthropic / Google

| | Centralized AI (OpenAI, etc.) | BTCPC |
|---|---|---|
| Who runs it | One company | A network of independent miners |
| Single point of failure | Yes — one API outage affects everyone | No — if one miner goes down, others take over |
| Censorship | Provider decides what you can ask | Nobody controls what gets computed |
| Pricing | Corporate monopoly pricing | Market-driven, competitive |
| Privacy | Your prompts go through corporate servers | Distributed — no single entity sees all traffic |
| Ownership | They own the infrastructure | Miners own their own hardware |

## The Name

- **BTC** — Bitcoin. The gold standard of decentralized networks.
- **PC** — Proof of Compute. Mining by doing useful computation.
- **42** million total supply — The Answer to the Ultimate Question of Life, the Universe, and Everything (Douglas Adams).
- **Dreams** — the smallest unit of BTCPC (like satoshis for Bitcoin). Every token is a dream computed into reality.

BTCPC is live and mining. The chain has been running continuously since April 2, 2026. There are currently 2 mining nodes producing blocks every 5 minutes, with 12 projects actively paying for AI inference on the network.
